Google (and others) stopped indexing sites automatically

June 29, 2023

Before here was my rambling about how google and other search engines are not indexing this site. But I wrote that I did my initial research. But now I did a deep dive into all SEO stuff and found that the theme that I use had a metatag

<meta name="robots" content="noindex, nofollow">

and required special configuration for it to change to ‘index, follow’. So I will change it today and see give it more time for search engines to catch up.

So currently it looks like that it was me who messed up. If you are currious abotu what was written here before, you can go into git log.

Hugo

For my current setup I use Hugo as a static site builder. And the theme that I use has a setting which puts ’nofollow’ if the site is not in production. And of course I did a mistake. So in case you are also using hugo and google search is important for youu then don’t forget to check your site for these types of silly errors.
